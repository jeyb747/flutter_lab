¿Qué es Flutter?

Es un framework o kit de desarrollo de software (SDK) creado por Google para diseñar aplicaciones móviles, web y de escritorio nativas a partir de un único código base.

SDK

Contiene todas las herramientas, compiladores y bibliotecas necesarias para desarrollar aplicaciones multiplataforma sin problemas de compatibilidad.

Desarrollo Mobile

Permite compilar directamente a código nativo tanto para iOS como para Android, asegurando un rendimiento óptimo y fluido.

Dart

El lenguaje de programación optimizado para el cliente que utiliza Flutter. Es fácil de aprender, rápido y soporta recarga en vivo (Hot Reload).

Comandos eficientes

Herramientas de terminal potentes (como flutter doctor o flutter run) que agilizan el proceso de desarrollo y diagnóstico.

El Universo de los Widgets

En Flutter, todo es un widget. Los widgets son los bloques de construcción básicos de la interfaz de usuario.

StatelessWidget

Son widgets sin estado. No cambian de forma dinámica una vez que se destruyen o renderizan. Se usan para elementos estáticos como textos, íconos o botones simples.

StatefulWidget

Son widgets con estado. Tienen una estructura interna que puede cambiar durante el ciclo de vida de la aplicación debido a interacciones del usuario o datos nuevos.
