# Taller Flutter: Introducción al Desarrollo Mobile

¡Bienvenido al repositorio del **Taller de Flutter**! Este proyecto es una página web informativa construida con HTML5 y CSS3 que recopila los conceptos fundamentales discutidos en la sesión del **29 de mayo de 2026**.

El objetivo de este sitio es servir como guía de estudio rápida para entender el ecosistema de Flutter, su lenguaje (Dart) y la arquitectura básica de widgets.

##  Contenido de la Página

La página explica de forma interactiva y visual los siguientes bloques temáticos:

1. **¿Qué es Flutter?**: Introducción al SDK de Google para el desarrollo mobile multiplataforma.
2. **Conceptos Clave**:
   * **SDK**: Herramientas y compiladores.
   * **Desarrollo Mobile**: Compilación nativa para Android e iOS.
   * **Dart**: Características del lenguaje optimizado para clientes.
   * **Comandos eficientes**: Automatización y diagnóstico en la terminal.
3. **El Universo de los Widgets**: 
   * `StatelessWidget` (Widgets sin estado).
   * `StatefulWidget` (Widgets con estado dinámico).
4. **Entorno de Trabajo**: Pasos iniciales utilizando **GitHub Codespaces** para los laboratorios (`Flutter_lab`).

---

##  Tecnologías Utilizadas

* **HTML5**: Estructuración semántica del contenido.
* **CSS3**: Diseño moderno, uso de CSS Grid, Flexbox, variables nativas y diseño responsivo (adaptable a móviles).
